News Article Analyzer for Stocks

This project is coded in Python and utilises streamlit. The app performs NLP (Natural Language Processing) analysis on the selected stock based on the latest news articles. It utilizes Google News and the VADER sentiment analysis tool to extract and analyze news sentiment. The app displays a dataframe of news articles, sentiment analysis results, and a pie chart illustrating the sentiment distribution (BUY, THINK, SELL). Additionally, it generates a word cloud visualization for the stock's news summaries. Users can input the company name to retrieve and analyze relevant news. It was created with assistance from the Greater Manchester AI Foundry (GMAI Foundry).

Time Series Forecasting for Stocks

Also created with assistance from the GMAI Foundry, this Python code creates an interactive AI Stock Forecaster app using Streamlit. The app allows users to select a stock from a predefined list, analyzes its historical data, and uses the Prophet library for time series forecasting. The forecasted data, including predicted values and confidence intervals, is presented through interactive Plotly charts. The app also displays raw data, forecasted components.
